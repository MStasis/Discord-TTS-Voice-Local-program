Voiceboard 설치 및 사용 방법
============================

이 프로그램은 Discord 봇 없이 사용자의 Windows PC에서 TTS와 사운드보드 소리를 Discord 음성 채널로 보내기 위한 로컬 데스크톱 앱입니다.


1. 준비물
--------

필수:
- Windows 10 또는 Windows 11
- 인터넷 연결
- Discord 데스크톱 앱
- VB-CABLE 가상 오디오 케이블

권장:
- Discord 입력 장치를 "Default"로 설정
- 마이크와 스피커가 정상 연결된 상태

VB-CABLE 공식 다운로드:
https://vb-audio.com/Cable/


2. VB-CABLE 설치 방법
--------------------

1. 위의 VB-CABLE 공식 다운로드 페이지로 이동합니다.
2. Windows용 VB-CABLE Driver Pack을 다운로드합니다.
3. ZIP 파일을 압축 해제합니다.
4. 압축을 푼 폴더에서 VBCABLE_Setup_x64.exe를 마우스 오른쪽 클릭합니다.
5. "관리자 권한으로 실행"을 선택합니다.
6. Install Driver 버튼을 눌러 설치합니다.
7. 설치가 끝나면 Windows를 재부팅합니다.

설치 후 Windows 오디오 장치에 아래 두 장치가 보여야 합니다.
- 재생 장치: CABLE Input(VB-Audio Virtual Cable)
- 녹음 장치: CABLE Output(VB-Audio Virtual Cable)

프로그램 실행 시 VB-CABLE이 없으면 설치 안내창이 자동으로 뜹니다. 안내창의 "다운로드 페이지" 버튼을 누르면 공식 다운로드 페이지가 열립니다.


3. Voiceboard 설치 방법
----------------------

설치 파일을 사용하는 경우:
1. Voiceboard-Setup-0.1.0-x64.exe를 실행합니다.
2. Windows SmartScreen 경고가 뜨면 "추가 정보"를 누른 뒤 "실행"을 선택합니다.
3. 설치 위치를 선택하고 설치를 완료합니다.
4. 바탕화면 또는 시작 메뉴의 Voiceboard 바로가기로 실행합니다.

무설치 파일을 사용하는 경우:
1. Voiceboard-Portable-0.1.0-x64.exe를 원하는 폴더에 둡니다.
2. 파일을 더블클릭해서 실행합니다.
3. Windows SmartScreen 경고가 뜨면 "추가 정보"를 누른 뒤 "실행"을 선택합니다.

참고:
- 이 앱은 별도의 Discord 봇 토큰이나 서버 초대가 필요하지 않습니다.
- 앱 데이터는 Windows 사용자 앱 데이터 폴더에 저장됩니다.
- 저장한 문장과 사운드 파일은 앱의 "보관함" 버튼으로 확인할 수 있습니다.


4. Discord 설정
---------------

권장 설정:
1. Discord를 엽니다.
2. 사용자 설정으로 들어갑니다.
3. 음성 및 비디오 메뉴를 엽니다.
4. 입력 장치를 "Default"로 설정합니다.

중요:
- Voiceboard의 "세팅" 버튼은 Windows 기본 녹음/통신 마이크를 CABLE Output으로 바꿉니다.
- Discord 입력 장치가 "Default"이면 Discord가 이 변경을 따라갑니다.
- Discord 입력 장치를 특정 마이크로 고정해 두면 Voiceboard가 자동으로 바꿀 수 없습니다.


5. 기본 사용 흐름
----------------

1. VB-CABLE을 설치하고 Windows를 재부팅합니다.
2. Discord 입력 장치를 "Default"로 설정합니다.
3. Voiceboard를 실행합니다.
4. 상단의 "세팅" 버튼을 누릅니다.
5. 출력 대상이 CABLE Input으로 바뀌었는지 확인합니다.
6. Discord 음성 채널에 들어갑니다.
7. TTS 문장을 입력하고 "실행"을 누르거나, 저장된 문장/사운드 버튼을 누릅니다.
8. 사용이 끝나면 "해제" 버튼을 눌러 기본 마이크로 되돌립니다.


6. 버튼 설명
------------

세팅:
- Voiceboard 출력 대상을 CABLE Input으로 설정합니다.
- Windows 기본 녹음/통신 마이크를 CABLE Output으로 설정합니다.
- Discord 입력 장치가 Default이면 Discord도 가상 케이블 입력을 사용하게 됩니다.

해제:
- 세팅 전의 Windows 기본 마이크로 복원합니다.
- Voiceboard 출력 대상을 시스템 기본 출력으로 되돌립니다.

장치 새로고침:
- 오디오 출력 장치 목록을 다시 읽습니다.
- VB-CABLE 설치 후 앱을 재시작하지 않고 확인할 때 사용합니다.

보관함:
- 앱 데이터 폴더를 엽니다.
- 저장한 사운드 파일과 앱 설정 파일을 확인할 수 있습니다.

정지:
- 현재 재생 중인 TTS와 사운드를 모두 멈춥니다.


7. TTS 사용 방법
----------------

1. 말하기 영역에 문장을 입력합니다.
2. 엔진을 선택합니다.
   - Neural Online: 온라인 Microsoft Edge Neural TTS 계열 음성입니다. 기본 추천입니다.
   - Windows Local: Windows에 설치된 로컬 음성입니다. 인터넷이 불안정할 때 사용할 수 있습니다.
3. 목소리를 선택합니다.
   - 기본 추천: SunHi Korean Female
4. 필요하면 속도, 피치, TTS 볼륨을 조절합니다.
5. "실행"을 누릅니다.

자주 쓰는 문장 저장:
1. 문장을 입력합니다.
2. 버튼 이름을 입력합니다.
3. "문장 저장"을 누릅니다.
4. 저장된 문장은 "자주 쓰는 문장" 영역에 버튼으로 표시됩니다.

저장된 문장 삭제:
- 문장 카드의 휴지통 버튼을 누릅니다.


8. 사운드보드 사용 방법
----------------------

사운드 추가:
1. "사운드 추가" 버튼을 누릅니다.
2. mp3, wav, ogg, m4a, aac, flac 같은 오디오 파일을 선택합니다.
3. 추가된 사운드는 사운드보드 카드로 표시됩니다.

사운드 실행:
- 사운드 카드의 "실행" 버튼을 누릅니다.

사운드 삭제:
- 사운드 카드의 휴지통 버튼을 누릅니다.


9. 볼륨과 내 듣기
----------------

Discord 송출 볼륨:
- Discord로 보내지는 가상 케이블 쪽 소리 크기만 조절합니다.
- 상대방에게 들리는 TTS/사운드보드 볼륨을 낮추거나 높일 때 사용합니다.

내 듣기:
- 켜져 있으면 같은 소리를 내 기본 스피커로 따로 재생합니다.
- 이 소리는 Discord 송출 볼륨의 영향을 받지 않습니다.
- 즉, 상대방에게 보내는 소리만 줄이고 내 PC에서 듣는 소리는 그대로 둘 수 있습니다.

주의:
- Windows의 "이 장치로 듣기" 기능이나 VoiceMeeter 자체 모니터링으로 CABLE Output을 듣는 경우, 그 소리는 케이블 신호 자체를 듣는 것입니다.
- 그런 방식으로 듣는 소리는 Discord 송출 볼륨과 같이 줄어들 수 있습니다.
- Voiceboard 앱 안의 "내 듣기" 기능을 사용하는 것을 권장합니다.


10. 문제 해결
-------------

문제: 앱 실행 시 VB-CABLE 설치 안내창이 뜹니다.
해결:
- VB-CABLE이 설치되어 있지 않거나 장치가 비활성화된 상태입니다.
- 안내창의 다운로드 페이지 버튼으로 공식 사이트에 들어가 설치합니다.
- 설치 후 Windows를 재부팅합니다.
- 앱에서 "장치 새로고침" 또는 안내창의 "다시 확인"을 누릅니다.

문제: Discord에 소리가 들어가지 않습니다.
확인:
- Voiceboard 출력 대상이 CABLE Input인지 확인합니다.
- Discord 입력 장치가 Default인지 확인합니다.
- "세팅" 버튼을 다시 눌러 봅니다.
- Discord 음성 및 비디오 설정에서 입력 감도가 너무 낮지 않은지 확인합니다.

문제: 내 스피커에서는 들리는데 Discord에는 안 들립니다.
확인:
- 내 듣기는 기본 스피커로 따로 재생됩니다.
- Discord로 보내려면 출력 대상이 CABLE Input이어야 합니다.
- Discord 입력은 CABLE Output 또는 Default여야 합니다.

문제: Discord에는 들리는데 나는 안 들립니다.
해결:
- Voiceboard의 "내 듣기" 체크가 켜져 있는지 확인합니다.
- Windows 기본 스피커가 정상인지 확인합니다.

문제: TTS 생성 실패가 뜹니다.
해결:
- Neural Online은 인터넷 연결이 필요합니다.
- 인터넷 연결을 확인하거나 Windows Local 엔진을 사용해 봅니다.

문제: Windows SmartScreen 경고가 뜹니다.
설명:
- 개인 배포용 앱이라 별도 코드 서명 인증서가 없으면 SmartScreen 경고가 뜰 수 있습니다.
- 파일 출처를 확인한 뒤 "추가 정보" > "실행"을 선택하면 됩니다.


11. 친구에게 전달할 때
---------------------

가장 쉬운 전달 방식:
- Voiceboard-Setup-0.1.0-x64.exe
- 설치및사용방법.txt

무설치로 전달할 때:
- Voiceboard-Portable-0.1.0-x64.exe
- 설치및사용방법.txt

친구에게 꼭 알려줄 점:
- VB-CABLE을 먼저 설치해야 합니다.
- 설치 후 재부팅하는 것이 좋습니다.
- Discord 입력 장치는 Default로 두는 것이 가장 편합니다.
- 사용이 끝나면 Voiceboard의 "해제" 버튼을 눌러 기본 마이크로 되돌리면 됩니다.
